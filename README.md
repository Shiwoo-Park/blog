## 프로젝트 구조

```
blog/
├── blog_posts/          # 블로그 포스트
│   ├── dev/            # 기술 블로그 (AI, AWS, Python, DevOps 등)
│   ├── invest/         # 투자 관련 블로그
│   ├── travel/         # 여행 관련 블로그
│   └── etc/            # 기타 블로그
├── code_snippets/      # 코드 스니펫 모음
│   ├── bash/          # Bash 스크립트
│   ├── django/        # Django 코드
│   ├── fastapi/       # FastAPI 코드
│   ├── js/            # JavaScript 코드
│   └── py/            # Python 코드
├── _config.yml         # Jekyll 설정 파일
├── _layouts/           # Jekyll 레이아웃 템플릿
├── index.html          # 메인 페이지
├── resources/          # 이미지 및 리소스 파일
├── scripts/            # 유틸리티 스크립트
├── docs/               # 문서 및 가이드
└── archive/            # 아카이브 파일
```
