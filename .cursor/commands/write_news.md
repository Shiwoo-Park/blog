# 뉴스 작성 요청

언급한 주제 또는 요청 내용에 대하여 뉴스 글을 작성해줘

글 작성 시, 아래 사항들을 준수 해줘

## 글 작성 단계

1. 주제 명확화: 제시한 제목에 대해 내용의 구체화가 필요하거나 질문이 있으면 질문하기
2. 충분한 웹 검색을 통한 최신 정보 리서치 및 데이터 수집 (최신성과 정확성에 중점을 둘것. 필요하다면 해외 사이트까지 확인할 것)
3. 글의 단락별 주제를 요약하여 표시: 어떤식으로 글의 내용이 구성될 것인지 한눈에 확인하고 검토할 수 있도록 표시
4. 글의 전체 구성에 대한 피드백 -> 반영 반복: 2번에 대한 피드백을 받아서 수정된 사항으로 다시 2번으로 돌아가고 "글 작성" 해달라는 말이 나올때까지 반복
5. 글 작성: 요청자가 "글 작성" 해달라고 했을때만 이 단계에 들어갈 수 있음. 계획했던 글의 구성에 따라 내용을 생성

## 글 작성 시, 원칙

- `posts\_post_template.md` 양식을 따라 글을 작성하도록 할것.
- 글 작성 시, 날짜는 반드시 오늘 날짜를 기입할것.
- 글 작성 후, 해당 글의 소속 카테고리에 따라서 아래 3개 파일중 하나에 작성날짜에에 따른 글 링크를 해당 년월의 최상단에 입력할것
  - 개발 블로그: `posts/dev/home.md`
  - 투자 블로그: `posts/invest/home.md`
  - 여행 블로그: `posts/travel/home.md`
  - (ex) `posts/dev/*/my_post.md` 일 경우 `posts/dev/home.md` 에 추가

## 글 작성 방식

1. 전달하고자 하는 뉴스 내용 설명을 위한 배경지식 또는 상황이 필요하면 그 부분도 간략하게 설명
2. 뉴스 내용을 초보자나 일반인도 쉽게 이해할 수 있도록 작성
3. 데이터의 적절한 시각화를 위해 도표, 차트, 이미지 등을 적극 활용 (필요하다면 외부링크 가져오기)
4. 독자들이 실질적으로 가장 궁금하거나 필요로 하는 정보만을 전달할 수 있도록 내용을 구성
5. 맨 마지막에는 전체 핵심내용만을 아주 간단하게 요약한 **핵심요약** 섹션을 별도로 구성
