## 재해장해 등급별 지급금액 도표

> 기준: **재해상해특약**
> 장해율 **3% 이상 ~ 80% 미만** → 장해급여금
> **80% 이상** → 재해사망보험금으로 지급

| 장해율(등급) | 장해 상태 예시        | 지급금액                    |
| ------- | --------------- | ----------------------- |
| 80% 이상  | 사지 마비, 전신 중증 장해 | **30,000,000원 (재해사망금)** |

| 40%     | 한 다리 기능 중대 제한   | 12,000,000원             |

# 수술 1·2·3종 분류 + 지급금액 도표

> 기준: **수술특약 + 수술분류표(별표)**
> ✔️ 실제 지급금액 확정
> ✔️ 여러 수술 동시 시행 시 **가장 높은 1종만 지급**

## ⚠️ 꼭 알아야 할 핵심 포인트

* **일반암만 90일 면책**, 나머지는 즉시 보장
* 암 진단 → 입원 → 수술 **중복 수령 가능**
* 재해사망(3천만) + 암진단(2천만) **동시에 발생 불가** (사망 전 조건 필요)
* 수술은 **같은 날 동시에 여러 개 해도 최고 1건만 지급**
* 장해율은 **의사 진단서 + 장해분류표** 기준
* 장해는 **영구 장해**만 인정
* 수술은 **“치료 목적”**이어야 함 (미용·예방 목적 ❌)
* 수술은 질병·재해 모두 적용
* 암 수술은 **암수술급여금과 별도**로 중복 지급 가능
* 수술 + 암수술 + 암진단비 **중복 수령 가능**
